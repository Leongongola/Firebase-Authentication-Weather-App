## Introduction

MyWeatherApp is a simple web application developed to help users stay informed about the current weather conditions in different cities. Users can easily search for a city and view detailed weather information such as temperature, humidity, wind speed, and more.

## Features

- View real-time weather information for multiple cities
- Search for weather data by city name
- Display temperature in Celsius
- Responsive design for seamless use on various devices
